인터넷 -> 스마트폰->
데이터가 많이 모였다 -> 빅데이터 -> 데이터분석
인공지능
라이브러리가 많다. -> pandas, plotly, streamlit 등
수학관련 라이브러리가 많다. -> 인공지능 활성화
무료 , 오픈소스 (깃허브에 코드올리면 코드 공개)
종류가 많다. 인기에 휩쓸림
1.머신러닝 : 데이터에서 패턴을 학습 예측, 기계 학습
   지도학습(정답이 있다),비지도학습(정답이 없다, 예측), 강화학습

2.딥러닝 : 인공신경망, 이미지, 소리, 영상(비정형데이터)
   CNN(이미지),RNN(시퀀스), 트랜스포머(대규모 텍스트 데이터를 학습)....

3.LLM : 거대언어모델, 파라미터가 수십~수천억 이상
   -> 연산이 많이 들어간다 -> 병렬처리 -> GPU -> 일반인이 할 수 없다
    GPT, BERT, 라마
    챗GPT(2022년) -> 생성형AI

4.멀티모달 : 텍스트 + 이미지 + 소리 + 영상 동시에 처리하는 LLM
    -> GPT-4o,Gemini, Claude3

5.LAG -> Langchain (랭체인) : LLM 기능들을 모듈로 묶어 연결해서
    앱처럼 만들 수 있는 파이썬 프레임워크

6.LangGraph (랭그래프) -> 랭체인을 그래프구조로 확장
    워크플로우 자동화 프레임워크(쉽게 해준다)

7. Agent (에이전트) -> 비서, LLM이 도구를 선택하고 실행을 판단
    스스로 일처리하는 시스템

8.MCP -> 에이전트가 협업하는 프로토콜(통일된 규칙)

9.A2A -> 구글, 자연어로 소통
